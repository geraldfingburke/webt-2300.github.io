Promises Game

Design and build a trivia game that uses promise-based timing

Requirements

The application should utilize the Open Trivia API

The questions should be multiple choice and can be of any difficulty or category

You should create a generic recursive promise with a timeout that manages the gameplay loop

The player should press a button to start the game, then answer a series of questions (using buttons) with a time limit

The user should be able to change their answer at any point before the timer runs out and it should be clear which answer they have currently selected
