Final Project Rubric

Below are the following aspects I will be grading and their weights

The best way to use this rubric is to ask yourself each of these questions before finishing work on your project

Category

Description

Weight

Presentation
  • How does your application look? (UI)
  • How does your application feel? (UX)
  • Is the design cohesive?
 10%

Code
  • Is your code free of errors?
  • Is your code properly documented?
  • Do you make use of appropriate libraries and packages?
 30%

Project Structure
  • Do you use proper resource fetching (script and link tags for css and js)
  • Do you have an easy to navigate directory structure?
  • Is your application accesible from the web server?
 20%

API(s)
  • Does your application make use of one or more APIs?
  • Does your application dynamically change content based on API requests?
  • Does your application provide dynamic content upon user input?
 20%

Creativity
  • Does your application use interesting data in interesting ways?
  • Does your application show a sense of personal style?
  • Did building your application challenge you as a developer?
*Creativity is subjective, I am primarily evaluating the amount of effort you put into making something good 		20%
